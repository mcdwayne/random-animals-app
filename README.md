# 🐾 Random Animals App

A beautiful, modern web application that displays random animals from various public APIs. Discover amazing wildlife with just a click!

## ✨ Features

- **Random Animal Discovery**: Get random animals from multiple APIs
- **Specific Animal Types**: Choose between dogs, cats, and foxes
- **Beautiful UI**: Modern, responsive design with smooth animations
- **Favorites System**: Save your favorite animals locally
- **Statistics Tracking**: Keep track of animals viewed and favorites
- **🆕 Shopify Integration**: Sell beautiful prints of discovered animals
- **Shopping Cart**: Add prints to cart and checkout seamlessly
- **Multiple API Sources**: Uses various public APIs for variety
- **Fallback System**: If one API fails, automatically tries another
- **Responsive Design**: Works perfectly on desktop and mobile devices

## 🛍️ Shopify Integration

The app now includes a complete Shopify integration for selling animal prints:

- **Buy Print Button**: Hover over any animal image to see the buy button
- **Print Sizes**: Choose from 8"x10", 11"x14", or 16"x20" prints
- **Shopping Cart**: Add multiple prints to your cart
- **Checkout Process**: Seamless Shopify-powered checkout
- **Sales Tracking**: Monitor prints sold and revenue

## 🚀 How to Use

1. **Open the Application**: Simply open `index.html` in your web browser
2. **Get Random Animals**: Click the "Get Random Animal" button for surprise animals
3. **Choose Specific Types**: Use the dog, cat, or fox buttons for specific animals
4. **Save Favorites**: Click on the heart icon next to animal names to save favorites
5. **Buy Prints**: Hover over animal images and click "Buy Print" to add to cart
6. **View Statistics**: See how many animals you've discovered, saved, and prints sold
7. **Checkout**: Use the shopping cart to complete your purchase

## 🌐 APIs Used

- **Dog CEO API**: For random dog images
- **The Cat API**: For random cat images with breed information
- **Random Fox API**: For cute fox images
- **The Dog API**: Additional dog images with breed details

## 🛠️ Technical Details

- **Frontend**: Pure HTML, CSS, and JavaScript
- **No Dependencies**: No external libraries required (except Shopify SDK)
- **Local Storage**: Favorites and cart are saved in your browser
- **Responsive Design**: CSS Grid and Flexbox for modern layouts
- **Error Handling**: Graceful fallbacks if APIs are unavailable
- **Shopify Integration**: Uses Shopify Buy Button SDK for e-commerce

## 🔧 Shopify Setup

To enable the Shopify integration:

1. **Create a Shopify Store**: Set up your online store
2. **Enable Storefront API**: Generate access tokens
3. **Create Products**: Set up print products with pricing
4. **Update Configuration**: Add your Shopify credentials to `script.js`
5. **Test Integration**: Verify the buy buttons work correctly

📖 **Detailed Setup Guide**: See `shopify-config.md` for complete configuration instructions.
